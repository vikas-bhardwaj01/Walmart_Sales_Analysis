# Walmart Sales Analysis

## Overview

This project focuses on analyzing Walmart sales data to understand customer behavior, sales trends, and overall business performance. The analysis is based on solving real-world business problems using data-driven techniques.

Dataset was cleaned and preprocessed before analysis.

## Objectives

* Analyze payment methods and their impact on sales
* Identify highest-rated product categories across branches
* Determine busiest days and peak sales periods
* Evaluate sales performance by city and category
* Calculate profit and revenue trends
* Understand customer purchasing behavior

## Tools and Technologies

* Python (Pandas, NumPy, Matplotlib)
* SQL
* Jupyter Notebook

## Approach

The dataset was cleaned and structured before analysis. Python was used for data analysis and visualization, while SQL was used to solve business queries and extract insights.

## Key Insights

* Payment preferences vary across branches and influence transaction volume
* Some categories receive high ratings but do not generate the highest profit
* Sales patterns change based on day and time, indicating peak business hours
* Revenue contribution differs across cities and product categories

## Project Structure

* `data/` → cleaned dataset
* `notebooks/` → Jupyter Notebook analysis
* `sql/` → SQL queries
* `docs/` → business problem statements

## Conclusion

This project demonstrates how data analysis can be used to generate meaningful insights in a retail environment. The results can support better decision-making, improve sales strategies, and enhance overall business performance.
